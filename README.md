# Parking-Indicator-with-Matlab-and-Arduino-Integration
This program, developed using MATLAB, reports the occupancy status of indoor parking lots. The program includes integration with Arduino.
Materials used:

- Arduino Uno
- HC-SR04 Ultasonic Sensor
- 1602A Display with I2C Module

Circuit Connections:

HC-SR04 Ultrasonic Sensor Connections: 

VCC → Arduino 5V 
GND → Arduino GND 
TRIG → Arduino Digital Pin 9 
ECHO → Arduino Digital Pin 10 

1602A LCD + I2C Module Connections: 

VCC → Arduino 5V 
GND → Arduino GND 
SDA → Arduino Analog Pin A4 
SCL → Arduino Analog Pin A5
